---
title: 'Read Document'
description: 'Reads a document format input and extracts its textual content.'
---

The `Read Document` module is a data module that can read a document and extract all its textual content.

You can use this module to transform any document of `PDF`, `DOCX`, `TXT` into text.

The `Read Document` module has **one input** and **one output**:

- **Input**: One input of type `Document`
- **Output**: `Text`, the output is the text inside the document you provided
