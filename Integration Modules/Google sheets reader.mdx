>**This Module requires that you connect your account in the `Linked accounts` page. Go to Settings to get this done.**

<img height="200" src="GSR.png" />

The `Google sheets reader` module allows you to read data from a google sheets. 

**Google sheets are processed by column, you have to maintain the first row of the google sheets for the headers where each column represents a variable.**

The `Google Sheets Reader` module has **variable outputs**:

- **Outputs**: Each column header in the first row of the google sheets represents a `variable output` that will have a list composed by all the rows in each column.
- **Config**: The google sheets file can be loaded with a link or via the google finder
